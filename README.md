This project allows you to see the details of a pixel at the current location of the mouse pointer. The current build provides mouse pointer coordinates, RGB values, HSV values, Hex values, and a panel for you to view the pixel color in a larger area.

<img alt="Demo" src="https://user-images.githubusercontent.com/22604455/43696005-92264226-9901-11e8-849a-68ec2249882a.PNG" width="100%"/>

## Building
This project is built with [Maven](https://maven.apache.org/). To build the `pom.xml`, please follow the import instructions for your IDE.
* [IntelliJ](https://www.tutorialspoint.com/maven/maven_intellij_idea.htm)
* [Eclipse](https://www.tutorialspoint.com/maven/maven_eclispe_ide.htm)
